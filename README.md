# galileoff-InteractiveScraper
Web tabanlı etkileşimli bir scraping platformudur. Dashboard üzerinden taranan sitelerden çekilen veriler toplanır, kategorize edilir ve görselleştirilir. Anonim veri toplama ihtiyacı için TOR üzerinden scraping yapan bir modül içerir. Ölçeklenebilir, kontrol edilebilir ve merkezi veri izleme odaklı tasarlanmıştır.
